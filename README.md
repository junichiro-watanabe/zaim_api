## 目的
zaim の API を利用するサンプルコードを ruby で書く。

## 各ファイル説明
- sample_code.rb

ruby でのメイン処理を記載

- sample_code.html.erb

結果表示用の ruby 埋め込み html ファイル

- webrick.rb

webrick でのサーバ立ち上げ用ファイル

- .rubocop.yml
- .rubocop_todo.yml

rubocop でコード解析したため、rubocop 関連の設定ファイル

- Dockerfile
- docker-compose.yml

docker で開発環境を整えたため、docker 関連ファイル

- Gemfile
- Gemfile.lock

Gem 関連ファイル
